# emb3r
A modbus fan programing library with a familiar structure, with intent to make programing fans easier for everyone. 

Following stuctures of past projects, this project is intended to program emb_pabst fans, without having to use the OEM software. 
ECisCOOL? We Dont think so. We are desprately searching for an autmation friendly programing interface, and we are not opposed to 
programing it ourselves.
